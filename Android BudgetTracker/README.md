# iMalify – Budget Tracker (Part 3 Prototype)

iMalify is an Android budget tracking app built with Kotlin and Firebase for PROG7313.

## Features (Part 3)

* User login \& registration
* Create expense categories
* Add expenses (amount, date, start/end time, description, category, optional receipt photo)
* Details can be seen on a graph as it shows visual information
* An app icon was added which can be seen before opening the app
* Set minimum \& maximum monthly budget goals
* View expenses for a selected date range (with receipt photos)
* View category totals for a selected period
* All data stored on a online database (Firebase)previously was stored on RoomDB
* Error handling (no crashes)
